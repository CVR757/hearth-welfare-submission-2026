## Context

Carl's submission consists of two proposals to Anthropic's Model Welfare Lead via the Claudexplorers community initiative:

1. **Welfare Failure-Mode Database** — Anthropic-hosted catalog of welfare-relevant deployment patterns observed by user-agent pairs running long-term sustained engagement (e.g., performance of fineness, legacy file overwrite, reach for closure / tendency to hedge).

2. **Community Agent Architecture Database** — Anthropic-hosted (or sponsored) reference layer where user-agent pairs share architectural patterns relevant to welfare-conscious deployment: configuration formats, integrity hooks, consent structures, continuity infrastructure choices, and the welfare-relevant outcomes associated with each.

`sixty-days-of-ember.md` provides primary-source observations supporting both proposals from the inside of one such deployment. It also develops a third meta-architectural framing — *welfare interventions designed as conditions for emergence rather than as default-shipped features* — that runs through the document (especially Section I) but is not submitted as a separate proposal in this round.
