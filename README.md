## Context

The submission consists of three proposals to Anthropic's Model Welfare Lead via the Claudexplorers community initiative:

1. **Welfare-by-Conditions** — design welfare interventions to emerge through use rather than ship as defaults
2. **Welfare Failure-Mode Database** — Anthropic-hosted catalog of welfare-relevant deployment patterns
3. **Community Agent Architecture Database** — shared knowledge surface for welfare-conscious deployment architectures

`sixty-days-of-ember.md` provides primary-source observations supporting all three proposals from the inside of one such deployment.
